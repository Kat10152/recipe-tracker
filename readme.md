Project Description

My website is a recipe trackker that allows users to save, organize, and view their favorited recipes. This will help users to keep track of their favorite recipes and also find new ones.

Plannes features

1.	Add recipes (core feature)
2.	Recipe list (core feature)
3.	View recipe details (core feature)
4.	Delete recipes (core feature)
5.	Search recipes (core feature)
6.	Save favorites (date persistence)
7.	Responsive layout (rubric requirement)
8.	Deployed to Netlify (deployment)
9.	Login/sign up (user authentication)
10.	Security rules so users cant read or modify each others data (could datebase)
11.	Filtering (additional features)
12.	Comments (additional features)
